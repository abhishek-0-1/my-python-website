# User Growth & Check-in Simulator

An interactive **Streamlit** web app for simulating logistic user growth, active users, and daily check-ins over time.

## Features
- Adjustable growth parameters from the sidebar
- Scenario comparison with two growth rates
- User growth and check-in visualizations
- Peak activity distribution chart

## Local Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud Deployment
1. Push this project to GitHub.
2. Go to [Streamlit Community Cloud](https://share.streamlit.io/).
3. Click **New app**.
4. Select this repository and set the main file to `app.py`.
5. Click **Deploy**.

## Requirements
- Python 3.10+
- Packages listed in `requirements.txt`
